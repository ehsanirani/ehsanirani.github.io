Modelling Bacteria Migration in Porous Media
Posted on 2025-07-24 :: active-matter

This project began as a hobby side-project to model bacteria as active rod-shaped agents and study their migration in porous environments. Some of this work has been published in Phys. Rev. Lett. 128, 144501.
